# Nice Gadgets Store (React E-commerce)
A modern e-commerce web application built with React and TypeScript.
The project simulates an online gadget store with product browsing, filtering, cart functionality, and a clean user interface.

## 🔗 Live Preview
[View Live Demo](https://<your-vercel-link>.vercel.app)

## 🎯 Project Overview
This project represents a fully functional frontend e-commerce application.
Users can browse products, view detailed pages, add items to cart, and navigate through categories.

## 🚀 Getting Started
Follow these steps to run the project locally:

### 1. Clone the repository
git clone https://github.com/Vladislav-Korsun/react-ecommerce-store.git

### 2. Navigate to the project folder
cd react-ecommerce-store

### 3. Install dependencies
npm install

### 4. Run the project
npm run dev
